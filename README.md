# Parking Lot Ticketing System (C Project)

A simple parking management program in C that stores vehicle entry/exit details, calculates parking fee and generates daily reports. Uses file handling to save records even after program closes.

---

## Features
- Add vehicle (number + type)
- Exit vehicle with fee calculation
- View currently parked vehicles
- Daily report (2W/4W + revenue)
- First 30 minutes free

---

## Technologies Used
C Language, File Handling, Structures, Functions, time.h
